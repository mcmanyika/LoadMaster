### Pricing Tiers Breakdown

#### Essential Plan
- **Monthly**: $99/month
- **Annual**: $85/month (15% discount) = $1,020/year
- **Target**: Small fleets (1-5 trucks)
- **Features**: Up to 5 users, 50 loads/month, basic features

#### Professional Plan
- **Monthly**: $199/month
- **Annual**: $170/month (15% discount) = $2,040/year
- **Target**: Growing fleets (5-20 trucks)
- **Features**: Up to 15 users, 500 loads/month, advanced features

#### Enterprise Plan
- **Pricing**: Custom (typically $500-$2,000+/month)
- **Target**: Large operations (20+ trucks)
- **Features**: Unlimited users, unlimited loads, custom integrations
